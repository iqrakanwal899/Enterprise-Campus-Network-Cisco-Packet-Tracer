## Overview
This project implements an **Enterprise Campus Network** using **Cisco Packet Tracer**, designed to interconnect multiple university campuses through a central core router. The network supports large numbers of users with scalable and reliable connectivity.

## Network Type
**Enterprise / University Campus Network with WAN Connectivity**

## Architecture
- **Core Layer:** Central router providing inter-campus routing  
- **Distribution Layer:** One router per campus  
- **Access Layer:** Multiple Cisco 2960 switches connecting end devices  

## Connected Campuses
- BU H-11 Campus  
- BU E-8 Campus  
- BU Karachi Campus  
- BU Lahore Campus  

## Key Features
- Hierarchical network design  
- LAN and WAN integration  
- High user density support  
- Scalable and expandable topology  

## Tools Used
- Cisco Packet Tracer  
- Cisco Routers & 2960 Switches
